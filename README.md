🚀 VK Ultimate

VK Ultimate — это мощное десктопное приложение для автоматизации, мониторинга и управления аккаунтом ВКонтакте. Программа объединяет в себе функции "умного" лайкера, шпиона за группами, монитора новостей и полноценного чат-клиента в современном интерфейсе.


✨ Основные возможности:

🛠 Автоматизация (Лайкер)

    Массовый лайкинг: Простановка лайков на посты выбранного пользователя или группы.

    Режим удаления: Возможность снимать свои лайки (Unlike).

    Фильтры: Настройка глубины поиска, лимита лайков, фильтрация по дате (день/месяц/год).

    Безопасность: Игнорирование постов со «стоп-словами» (например: траур, погиб, продам).



📡 Мониторинг (Sentinel & News)

    Монитор групп (Sentinel): Отслеживание новых постов в выбранных пабликах в реальном времени. Уведомление приходит мгновенно.

    Монитор новостей: Поиск постов во всей ленте новостей по ключевым словам. Найденные посты сохраняются в Архив.

    Работа в фоне: Программу можно свернуть в системный трей, мониторинг продолжится.



💬 Общение и Медиа

    Встроенный Чат: Полноценное окно переписки с поддержкой вложений (фото), эмодзи, историей сообщений и поиском по дате.

    LikeMe: Просмотр последних активностей (кто лайкнул вас, кто прокомментировал).

    Парсер: Поиск старых постов, аудио или видео на стене пользователя за конкретную дату.

⚙️ Особенности интерфейса

    Современный дизайн на CustomTkinter (Тёмная/Светлая тема).

    Звуковые уведомления (mp3) для разных событий.

    Менеджер контактов для быстрого переключения между целями.
	
	
📦 Установка и Запуск

    Распакуйте программу в удобное место.
	
	Файлы (словари, адресная книга и прочее) хранятся в папке "Мои документы"
	
	!Внимание там же лежит токен для доступа. Будте осторожны не отдайте его вместе с программой!
	
	
🔑 Настройка (Токен)

    Для работы программы требуется Access Token от ВКонтакте.

    Нажмите кнопку "Сменить токен" в программе (или перейдите на vkhost.github.io).

    Выберите приложение (например, Kate Mobile или VK Admin).

    Разрешите доступ и скопируйте все из адресной строки в окно программы.

    Он сохранится локально в файле vk_token.txt.
	
	
⚠️ Дисклеймер

    Программа использует неофициальное взаимодействие с API ВКонтакте. Разработчик не несет ответственности за возможные блокировки аккаунта при превышении лимитов (спаме).

    Рекомендуется использовать задержки (встроены в программу).

    Не ставьте более 500 лайков в сутки.
