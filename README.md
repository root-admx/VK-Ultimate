# VK-Ultimate
VK Ultimate — это мощный комбайн для автоматизации действий, сбора информации и парсинга контента в социальной сети ВКонтакте. Программа объединяет в себе функции «умного» лайкера, парсера постов, загрузчика медиа-списков и инструмента для глубокой аналитики профилей.


🔥 Возможности
1. 🤖 Умный Лайкер (Mass Liker)

    Автоматизация: Ставит или удаляет лайки на постах пользователей и групп.

    Фильтр «Стоп-слова»: Пропускает посты с плохими новостями (например, "траур", "сбор средств") или рекламой.

    Гибкие настройки: Установка глубины поиска и лимита действий.

    Имитация человека: Случайные задержки между действиями для защиты от бана.

2. 📅 Машина Времени (Smart Parser)

    Бинарный поиск: Уникальный алгоритм, который находит посты за 2015 год (или любой другой) за пару секунд, не перебирая всю ленту.

    Фильтры даты: Поиск по конкретному дню, месяцу, году или за всё время.

    Результат: Сохраняет ссылки и тексты постов в файл + открывает интерактивное окно с кнопками "Перейти".

3. 🎵🎬 Медиа Парсер

    Видео: Собирает список видео со стены и из видеозаписей профиля.

    Аудио: Находит музыкальные треки, прикрепленные к постам на стене.

    Экспорт: Создает отчет со ссылками на медиа-контент.


4. 🛠 Установка и Запуск

    Просто запустите файл VK Ultimate.exe.
   
	
5. 🔑 КАК ПОЛУЧИТЬ ТОКЕН (Обязательно к прочтению)

    Для работы программы требуется Access Token (ключ доступа). Это разрешение от ВК на управление вашим аккаунтом через программу.

    Инструкция:

    Нажмите в программе кнопку "Сменить токен".

    Откроется сайт vkhost.github.io (это надежный сервис для получения токенов официальных приложений).

    На сайте выберите приложение Kate Mobile (рекомендуется) или VK Admin.

    Нажмите "Разрешить" (Allow).

    После успешного входа вы увидите белую страницу, а в адресной строке браузера будет длинная ссылка.

    Скопируйте эту ссылку целиком (она начинается на https://oauth.vk.com/...).

    Вставьте ссылку в появившееся окно программы и нажмите OK.

    Важно: Токен сохраняется в файл vk_token.txt рядом с программой. Никому не передавайте этот файл!

6. 🔎 Как искать пользователей

    В поле "Цель" вы можете вводить данные в любом формате:

    ID: 1 (для Павла Дурова) или id1.

    Короткое имя: durov или apiclub.

    Ссылка: https://vk.com/durov или https://vk.com/public1.

    Для групп: Можно использовать ID группы, например club12345 или public12345.

    Совет: После ввода нажмите Enter или кликните в любое место — программа подгрузит аватарку и имя, чтобы вы убедились, что цель выбрана верно.

7. ⚠ Меры предосторожности (Anti-Ban)

    ВК борется с автоматизацией. Чтобы ваш аккаунт не заморозили, соблюдайте правила:

    Не жадничайте: Лимит ВК — 500 лайков в сутки. В программе рекомендуем ставить не более 150–200 лайков в день.

    Не спешите: В программе встроены задержки (4–8 секунд). Не пытайтесь их убрать в коде, если не понимаете рисков.

    Не спамьте: Не запускайте "Парсинг за все время" на аккаунтах с 100 000 постов слишком часто.

    Стоп-слова: Используйте их, чтобы не лайкать траурные посты или конкурентов.
